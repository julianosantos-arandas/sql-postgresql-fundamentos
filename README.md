# SQL - PostgreSQL — Exercícios

Este repositório contém exercícios práticos de SQL desenvolvidos com
PostgreSQL, com foco em fundamentos da linguagem e organização do raciocínio.

Os exercícios foram executados utilizando pgAdmin e documentados em arquivos
`.sql` e `.md` para fins de estudo e portfólio.

## Tecnologias
- PostgreSQL
- pgAdmin
- SQLhttps://github.com/julianosantos-arandas/sql-postgresql-fundamentos/blob/main/README.md

## Estrutura do repositório
- `DESAFIO_SQL/`: estudo de caso baseado em SQL
- `comamdos_sql/`: consultas SQL organizadas por tema
- `imagens/`: prints dos resultados para base de links na pasta `SQL/`

## Dataset utilizado
- AdventureWorks
